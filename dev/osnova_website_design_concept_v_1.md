# Концепция дизайна сайта ОСНОВА ИТ / OSNOVA

## 1) Главная идея
Сайт должен транслировать ощущение **интеллектуальной технологической платформы enterprise-уровня**:

- современно
- визуально легко
- премиально
- инженерно точно
- эмоционально «умно»

Ключевая метафора: **«нервная система бизнеса»**.

Визуально это выражается через:
- плавные линии связей
- графовые анимации
- «живые» data-потоки
- модульные блоки
- архитектурную сетку
- мягкое свечение AI-узлов

---

## 2) Арт-направление
Смешиваем 3 слоя:

### A. Технологический модерн
- glassmorphism (очень умеренно)
- soft gradients
- ambient glow
- micro-interactions
- premium enterprise minimalism

### B. Классика искусства
Опора на:
- баухаус (структура)
- конструктивизм (ритм блоков)
- швейцарская школа (типографика)
- русский авангард (акцентные композиции)

### C. AI-native motion
- живые нейронные линии
- flowing particles
- node pulses
- intelligent hover states

## 6) Motion & эффекты
## Обязательно

### A. Scroll storytelling
При скролле пользователь буквально «путешествует» по системе.

### B. Node animation
Узлы системы пульсируют, показывая живую работу.

### C. Hover physics
Карточки слегка реагируют на курсор.

### D. Performance-first
Все эффекты только GPU-friendly:
- transform
- opacity
- blur минимально
- canvas particles только локально

Сайт должен быть **очень быстрым**.

## 8) UX-принцип
Главная цель:
> за 30–40 секунд объяснить CTO или директору, почему OSNOVA решает их хаос.

Поэтому UX должен:
- быстро вести к смыслу
- избегать перегрузки
- объяснять через визуальные потоки
- показывать value, а не только features

## 9) Техническая реализация
